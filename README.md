## 常见问题 FAQ

**搬瓦工日本东京 CN2 GIA 和大阪 CN2 GIA 有什么区别？**

线路性质一样，都是三网回程 CN2 GIA，延迟差距不大（通常 5-10ms 以内）。主要区别在价格和机房灵活性：东京贵将近一倍，但买了就是单一机房；大阪便宜，而且能在 KiwiVM 迁移到其他机房，选择空间更大。大多数预算有限的人选大阪更合算。

**The Tokyo Plan 限量版和东京 CN2 GIA 哪个好？**

两者线路不一样，不是同一个机房。东京 CN2 GIA 走电信 CN2 GIA 专线，线路等级更高，价格也贵得多。The Tokyo Plan 走 CMI 线路，对移动用户友好，价格便宜，但限量经常缺货。预算充裕选东京 CN2 GIA，想便宜就抢限量版，别纠结。

**搬瓦工日本站点支持解锁日本流媒体吗？**

东京 CN2 GIA（JPTYO_8）提供的是日本原生 IP，实测对日本本地流媒体服务解锁效果不错，YouTube 移动网络下播放 8K 也很顺畅。具体支持哪些平台取决于 IP 类型，数据中心 IP 和住宅 IP 解锁能力有差异，购买前可以参考官网说明或申请退款周期内测试。

**搬瓦工日本 VPS 适合建站吗？**

适合，尤其是目标用户在中国大陆或日本的站点。东京和大阪 CN2 GIA 机房延迟低、稳定性好，建站首选。如果预算有限，CN2 GIA-E 套餐选大阪软银机房也是不错的入门方案，$169.99/年，后期还能迁机房。

**如何判断我用的是哪个运营商？**

手机卡看套餐：中国电信（Chinanet）、中国联通（ChinaUnicom）、中国移动（ChinaMobile）。电信用户优先东京/大阪 CN2 GIA；联通用户优先大阪软银；移动用户东京限量版（CMI）效果最好，常规选项也可以选软银或 CN2 GIA。
